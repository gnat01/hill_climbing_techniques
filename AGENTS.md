## Hill climbing techniques

In this directory, our goal is to have a comprehensive suite of hill climbing techniques. The approach is to assemble the best papers, have Claude code read them end to end. Then :
For each paper, have its own min well-named directory. Within each :

- Synthesise synposes of each paper that gives a solid outline with required math of each paper
- Code up relevant algorithms from each paper in R and python along with any required visualisations
- Mention extensions

- Across the entire directory, have a logical (temporal) progression of ideas culminating in the state of the art.

## Coding instructions
- Codes should be ready to open PRs for and merge to main
- Codes must maintain high quality and be tested against a test suite

## Research ideation
- Where relevant, Claude Code should mention how ideas link up and why

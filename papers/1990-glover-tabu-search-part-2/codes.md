# Codes in Python and R

## What Can Be Coded

### Python

- Frequency-based long-term memory on top of a standard tabu engine.
- Elite-solution archives for intensification.
- Diversification by penalizing frequently used move attributes or components.
- Feasible-infeasible oscillation for constrained combinatorial problems.

### R

- Smaller-scale prototypes illustrating frequency memory and elite recombination.
- Parameter studies on tenure, diversification strength, and aspiration criteria.
- Visualization of attribute frequencies over time.

## Extensions

- Path relinking between elite solutions.
- Reactive memory adaptation.
- Multiobjective tabu search using multiple archives.
- Constraint-handling via adaptive penalties and oscillation.
- Learning-based scoring functions that replace hand-tuned diversification penalties.

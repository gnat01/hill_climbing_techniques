# Codes in Python and R

## What Can Be Coded

### Python

- An ILS framework with interchangeable local search, perturbation, and acceptance modules.
- TSP implementations using 2-opt local search and double-bridge perturbation.
- Basin-transition logging to estimate recurrence across local optima.
- Comparisons between ILS, random restart hill climbing, and SA.

### R

- Clear educational implementations for permutation and binary-string problems.
- Perturbation-strength sweeps with boxplots of final solution quality.
- Acceptance-rule experiments: improve-only, Metropolis-style, and record-to-record.

## Extensions

- Adaptive perturbation strength based on recent basin returns.
- Hybrid ILS with tabu or VNS local search.
- Population-based ILS maintaining multiple elite local optima.
- Multiobjective ILS with Pareto-archive acceptance.
- Learning perturbation operators from elite-solution differences.

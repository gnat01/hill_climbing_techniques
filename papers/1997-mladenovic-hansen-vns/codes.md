# Codes in Python and R

## What Can Be Coded

### Python

- Basic VNS and reduced VNS templates with pluggable neighborhood families.
- TSP implementations using 2-opt, insertion, 3-opt-style shaking, and segment perturbations.
- Knapsack or set-cover variants using increasing cardinality flips.
- Performance profiles across neighborhood orderings.

### R

- Small-scale VNS prototypes with clear experimentation loops.
- Visualization of neighborhood index over time and improvement events.
- Comparative studies against GRASP and tabu search.

## Extensions

- General VNS and skewed VNS variants.
- Adaptive neighborhood ordering based on recent gains.
- Hybrid VNS with reactive perturbation strength.
- Embedding exact subproblem solvers inside local search.
- Parallel VNS with multiple incumbents.

# Codes in Python and R

## What Can Be Coded

### Python

- A standard NSGA-II implementation for continuous or combinatorial problems.
- Fast nondominated sorting and crowding-distance routines.
- Visualization of evolving Pareto fronts on two- and three-objective benchmarks.
- Comparisons against scalarized hill climbing for multiobjective tasks.

### R

- Compact NSGA-II implementations on classical test suites such as ZDT functions.
- Pareto-front plotting and attainment-surface summaries.
- Experimental notebooks exploring population size and mutation strength.

## Extensions

- Hybrid NSGA-II with local search on selected archive members.
- Constraint-domination variants for constrained multiobjective optimization.
- Indicator-based survival criteria.
- Problem-specific crossover for routing and scheduling.
- Parallel evaluation and surrogate-assisted NSGA-II.

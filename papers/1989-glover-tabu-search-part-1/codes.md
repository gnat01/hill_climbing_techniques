# Codes in Python and R

## What Can Be Coded

### Python

- A generic tabu search framework with move attributes, tabu tenure, and aspiration criteria.
- Deterministic best-admissible-move selection for TSP, knapsack, or graph coloring.
- Logging of tabu hits, aspiration triggers, and cycle statistics.
- Experiments showing the difference between exact-state memory and attribute memory.

### R

- Clean implementations for subset-based and permutation-based problems.
- Comparative notebooks plotting trajectory quality against tabu tenure.
- Small educational examples demonstrating cycling under naive descent and its removal under tabu restrictions.

## Extensions

- Dynamic tenure rules.
- Candidate lists to reduce neighborhood evaluation cost.
- Strategic oscillation across feasibility boundaries.
- Long-term frequency memory layered on top of short-term tabu memory.
- Hybrid tabu plus local exact improvement operators.

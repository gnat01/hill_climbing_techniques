# Codes in Python and R

## What Can Be Coded

### Python

- Reactive tabu search for TSP, graph coloring, or MAX-SAT.
- Hash-based recurrence detection for visited solutions or move-attribute signatures.
- Dynamic tabu tenure updates and escape phases.
- Comparison against fixed-tenure tabu search under equal runtime budgets.

### R

- Prototype implementations that emphasize parameter traces and recurrence statistics.
- Visualizations of tenure adaptation over time.
- Reproducible experiments on small benchmark instances.

## Extensions

- Combine reactive tenure with long-term frequency memory.
- Learn recurrence thresholds from online performance.
- Apply reactive ideas to other metaheuristics such as variable neighborhood search.
- Use probabilistic tabu durations instead of deterministic ones.
- Add restart and elite-archive logic to create a fuller adaptive framework.

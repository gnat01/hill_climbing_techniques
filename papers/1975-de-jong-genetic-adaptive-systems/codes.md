# Codes in Python and R

## What Can Be Coded

### Python

- A canonical binary GA with configurable selection, crossover, and mutation.
- Reproduction of classic benchmark experiments on simple continuous test functions via binary encodings.
- Population-diversity and schema-frequency diagnostics.
- Comparisons against hill climbing and random restart on multimodal landscapes.

### R

- Educational GA implementations with strong statistical visualization.
- Parameter sweeps over mutation and crossover rates.
- Generation-by-generation plots of best, mean, and diversity measures.

## Extensions

- Real-valued encodings and modern variation operators.
- Hybrid GA plus local search to create a memetic algorithm.
- Constraint handling via penalty or repair operators.
- Multiobjective extensions leading toward NSGA-type methods.
- Landscape analysis comparing population search with local descent.
